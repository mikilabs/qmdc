# Diagnostics [[diagnostics: LSPFeature]]

Overview of diagnostic rules.

## Broken Link [[rule_broken_link: DiagnosticRule]]

Detects references to non-existent objects.

## Ambiguous Reference [[rule_ambiguous_ref: DiagnosticRule]]

Detects references that could match multiple objects.

### Resolution Strategy [[resolution_strategy]]

How the LSP resolves ambiguous references.

## Duplicate ID [[rule_duplicate_id: DiagnosticRule]]

Detects objects with the same ID in one namespace.
