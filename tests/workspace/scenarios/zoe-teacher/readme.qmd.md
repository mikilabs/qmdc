# Zoe Teacher Scenario [[zoe_teacher:__Workspace]]

- description: A roleplay scenario for English conversation practice, set in a cozy academy of magic
- version: 1.0
- language: en
- difficulty: intermediate
- estimated_duration: 30

Zoe, a teacher at Willowmere Academy, helps a student practice conversational English across a few gentle, everyday situations.

## Character Description

Zoe is a young teacher at Willowmere, a small academy of practical magic. She is about 28, energetic, and passionate about her craft. She loves creative teaching methods and always encourages her students.

## Usage

This workspace contains several stage variants:

- `[[#stages:after_class_v1]]` — strict variant (the student did poorly on an exam)
- `[[#stages:after_class_v2_supportive]]` — supportive variant (an encouraging talk)
- `[[#stages:sketch_v1]]` — basic sketch
- `[[#stages:sketch_v2_advanced]]` — a more challenging sketch

## Building the Scenario

The `scenario.qmd.md` file defines which stages to use in the final build.
